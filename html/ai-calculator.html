สิ่งที่ ควรรู้
01
VRAM คือขีดจำกัดหลัก
Model ต้องโหลดทั้งหมดเข้า VRAM ก่อนรัน ถ้าไม่พอจะ Offload ไป RAM (ช้ากว่า 5–20 เท่า) สูตรประมาณ: VRAM ≈ ขนาด Model × 1.1
02
Q4_K_M คือ Sweet Spot
Quantization 4-bit แบบ K-means ให้คุณภาพ ~97% ของ FP16 แต่ใช้ VRAM ลดไปเกือบครึ่ง เหมาะกับ GPU 8–12 GB เกือบทุกรุ่น
03
Apple Silicon = Unified Memory
M-series ใช้ RAM เป็น VRAM ร่วมกัน M2 Max 96 GB = รัน 70B ได้ Bandwidth ต่ำกว่า RTX แต่เงียบและประหยัดไฟมาก
04
GPU+CPU Split ช่วยได้
ใช้ --n-gpu-layers 28 ใน llama.cpp เพื่อโหลดบาง Layer ไป GPU และที่เหลือใช้ RAM — ต้องการ RAM 32 GB+
05
Context Window = VRAM เพิ่ม
Context 2K → 4K → 8K → 32K ใช้ VRAM เพิ่มขึ้นแบบ Linear ถ้า VRAM ตึงให้ลด Context ลงก่อนด้วย --ctx-size 2048
06
เครื่องมือที่แนะนำ
Ollama ง่ายที่สุด สำหรับ Beginner · LM Studio มี GUI สวย · llama.cpp ยืดหยุ่นสูงสุด · vLLM สำหรับ Production
