วิธีรัน Llama 3.1 8B บน PC
01
ติดตั้ง Ollama (ง่ายที่สุด)
ดาวน์โหลด Ollama จาก ollama.com แล้วรัน ollama pull llama3.1 — detect GPU อัตโนมัติ รองรับทั้ง NVIDIA CUDA และ Apple Silicon Metal
02
ใช้ LM Studio (มี GUI)
LM Studio รองรับ Llama 3.1 8B เลือก Quantization ได้ผ่าน UI ดาวน์โหลดจาก HuggingFace ในแอปได้เลย เหมาะกับผู้ที่ไม่ถนัด command line
03
Context Window 128K
Llama 3.1 8B รองรับ context สูงสุด 128K tokens แต่ context ที่ยาวขึ้นใช้ VRAM เพิ่ม ถ้า VRAM ตึงให้ตั้ง --ctx-size 4096 ใน llama.cpp เพื่อลด VRAM ใช้งาน
