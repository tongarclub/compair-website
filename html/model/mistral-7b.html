วิธีรัน Mistral 7B บน PC
01
ติดตั้ง Ollama (ง่ายที่สุด)
ดาวน์โหลด Ollama จาก ollama.com แล้วรัน ollama pull mistral — detect GPU อัตโนมัติ รองรับทั้ง NVIDIA CUDA และ Apple Silicon Metal
02
ใช้ LM Studio (มี GUI)
LM Studio รองรับ Mistral 7B เลือก Quantization ได้ผ่าน UI ดาวน์โหลดจาก HuggingFace ในแอปได้เลย เหมาะกับผู้ที่ไม่ถนัด command line
03
Context Window 32K
Mistral 7B รองรับ context สูงสุด 32K tokens แต่ context ที่ยาวขึ้นใช้ VRAM เพิ่ม ถ้า VRAM ตึงให้ตั้ง --ctx-size 4096 ใน llama.cpp เพื่อลด VRAM ใช้งาน
